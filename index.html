Autofocus Example

Click on the links below to see the different ways to set the autofocus attribute on an input element.

  • form + autofocus prop - doesn't work on iOS

  • document.getElementById('name').focus() + setTimeout - doesn't work on iOS

  • autofocus prop without form tag - doesn't work on iOS

  • document.getElementById('name').focus() + button - WORKS, like in the ss.ge

  • handleClick + queueMicrotask - WORKS

  • handleClick + setTimeout - doesn't work


The focus behavior functions correctly when minimizing and then reopening an already loaded page in all cases
